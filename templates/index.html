CFI Graph Visualiser

Operation time: 0 seconds

JSON Input Instructions:

1. Enter your graph as a JSON object or select an example

2. Format: {"node": ["neighbor1", "neighbor2", ...]}

3. Must include a node named 'x'

Visual Builder Instructions:

1. Click on the Add Node buttons to add a node

2. Click on the Add Edge button, then drag between two nodes to add an edge

3. Click on a node or edge to edit and/or delete it

4. Right-click a node (except 'x') to edit its name
